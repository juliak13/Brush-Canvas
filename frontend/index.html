Brush & Canvas

A database systems project designed to manage art courses, student registrations, and artwork submissions.

Project Overview

This project was developed as part of a Database Systems course and demonstrates SQL database design, relational tables, CRUD operations, and backend integration using PHP and SQLite.

Features

  • Student and instructor management
  • Course registration system
  • Artwork submission tracking
  • Relational database design
  • PHP backend with SQLite database

Technologies Used

  • HTML
  • CSS
  • PHP
  • SQLite
  • SQL
  • Git & GitHub
